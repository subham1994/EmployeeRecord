Technologies Used:
    - Django, Angular js, bootstrap, python 3.5
    - Application tested on: Firefox, Google chrome

Features:
    - Users can view all the companies listed on the website
    - Users can view all the employee for a particular company
    - Users can query for employees and companies using their name
    - Users can sort the columns based on various properties of employees and companies
    - Authenticated users add and delete company records
    - Authenticated users add, delete and modify company records
